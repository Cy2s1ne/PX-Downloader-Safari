## 中文

一个用于在 Safari 浏览器中下载 ph 站和 xv 站视频的 Safari 扩展。移植自 [Zgao](https://zgao.top) 的 Chrome 插件，并参考了 [xink2594](https://github.com/xink2594/PX-Downloader-Firefox) 的 Firefox 移植版本。

### ✨ 主要功能

- 自动识别并解析支持网站的视频源
- 提供多种清晰度视频下载选择
- 支持一键复制视频直链
- 简洁易用的操作界面

### 🔧 支持网站

- p\*\*\*hub.com
- xv\*\*\*\*\*.com

### 📥 安装方法

由于 Safari 扩展需要经过 Apple 签名，提供以下两种安装方式：

1. **从 Release 下载安装**：
   - 前往 [Releases](../../releases) 页面下载最新版本
   - 按照说明完成安装

2. **手动构建（开发者模式）**：
   - 克隆本仓库
     ```bash
     git clone https://github.com/Cy2s1ne/PX-Downloader-Safari.git
     ```
   - 使用 Xcode 打开项目并构建 Safari 扩展
   - 在 Safari 设置中启用"允许未签名扩展"（需要开启开发者模式）
   - 在 Safari 扩展管理页面中启用本扩展YOUR

### 📖 使用方法

1. 打开支持的视频网站并进入视频播放页面
2. 点击 Safari 工具栏中的扩展图标
3. 在弹出窗口中选择所需的视频清晰度
4. 点击下载按钮或复制直链

## English

A Safari extension for downloading videos from ph and xv sites. Ported from [Zgao](https://zgao.top)'s Chrome extension, with reference to [xink2594](https://github.com/xink2594/PX-Downloader-Firefox)'s Firefox port.

### ✨ Features

- Automatically detects and parses video sources on supported websites
- Offers multiple video quality options for download
- One-click copy of direct video links
- Clean and easy-to-use interface

### 🔧 Supported Sites

- p\*\*\*hub.com
- xv\*\*\*\*\*.com

### 📥 Installation

Since Safari extensions require Apple signing, two installation methods are provided:

1. **Download from Releases**:
   - Go to the [Releases](../../releases) page and download the latest version
   - Follow the instructions to complete installation

2. **Build Manually (Developer Mode)**:
   - Clone this repository
     ```bash
     git clone https://github.com/Cy2s1ne/PX-Downloader-Safari.git
     ```
   - Open the project in Xcode and build the Safari extension
   - Enable "Allow Unsigned Extensions" in Safari settings (requires enabling Developer Mode)
   - Enable the extension in Safari's extension management page

### 📖 Usage

1. Open a supported video website and navigate to a video page
2. Click the extension icon in the Safari toolbar
3. Select the desired video quality in the popup window
4. Click the download button or copy the direct link
